# 企业术语标准化检查

文档术语不一致，新人看得懵？
**标准化检查帮你统一术语。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 术语不再"乱用"

- **自定义字典**：JSON/YAML格式导入
- **批量处理**：最多20个文件同时检查
- **术语词典**：50+条常用术语参考
- **检查报告**：不规范术语高亮提示

## 怎么用

打开 `web_checker.html` → 上传文件 → 查看检查报告
